PasswordSafeLib

Reading Files.

The design of the API tries to make it easy for applications to read PasswordSafe databases.  To meet this aim, the constructors on PwsFile and its subclasses have been made non-public.  A factory method PwsFileFactory.loadFile(...) is used to load the files.  This method works out which version the file is and instantiates the correct subclass of PwsFile.

Adding Records

To add a new record to a database you must first create an empty record.  Calling newRecord() on the file instance creates a new empty record that is the correct subclass of PwsRecord.  The next step is to add data to the record.
